Point-in-Time Recovery (PITR)
La Rolls-Royce de la restauration
Composants
Sauvegarde complète de base
Journaux de transactions (binary logs)
Processus
1. Restaurer la dernière sauvegarde complète
2. Appliquer les journaux jusqu'à l'instant désiré
⚠ Sans les binary logs, la PITR est impossible. Toujours les activer en production !
